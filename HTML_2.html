Input Attributes

  1. value: specifies the initial value for an input field.

  2. readonly: specifies that the input field is read only.

  3. disabled: the input field is disabled.

  4. size: the size (how many characters) for the input field.

  5. maxlength: the maximum allowed length for the input filed.

  3. autofocus: the input field should automatically get focus when the page loads.

  4. form: specifies one or more forms an <input> element belongs to.
